Setup guide
1
Your Google Sheet CSV URL is already saved
The dashboard is pulling from your published Sheet every 60 seconds. If you ever change sheets, open this HTML file and update SHEET_CSV_URL at the top of the script tag.
2
Connect Tally to your Google Sheet
In your Tally form → Integrations → Google Sheets → Connect. Every EOD submission auto-appends a row. Make sure your form has fields for: Name, Calls made, Leads contacted, Deals closed, Revenue, Tasks completed, Notes.
3
Column name matching
The dashboard uses fuzzy matching to detect columns automatically. If something isn't showing up, check the yellow banner that appears on load — it shows exactly which headers were detected and what was matched. You can also edit COL_ALIASES in the script to fix any mismatches.
4
Share this dashboard
Go to app.netlify.com/drop → drag this HTML file in → get a free live URL in seconds (e.g. myteam.netlify.app). Share that link — anyone with it can view the live dashboard. It auto-refreshes every 60 seconds.
